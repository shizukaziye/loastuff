# loastuff

A single landing page linking every tool I've built for **Lost Ark** and **League of Legends**.

**Live:** https://shizukaziye.github.io/loastuff/

## Tools

**Lost Ark — Calculators**
- [Accessory Value Calculator](https://shizukaziye.github.io/lost-ark-accessories/) — score any accessory by its real % damage gain
- [Astrogem Calculator](https://shizukaziye.github.io/astrogem-calculator/) — cut/fuse/throw pipeline tables + screenshot advisor
- [Stronghold Crafting Profit](https://shizukaziye.github.io/loa-crafting-calculator/) — every craft ranked by net gold, gold/hr, ROI
- [Deal Finder](https://shizukaziye.github.io/loa-deal-finder/) — market items ranked vs a robust 14-day fair price

**Lost Ark — Community**
- [LOA ON Bingo](https://loa-on-bingo.shizukaziye.workers.dev/) — live multiplayer watch-party bingo
- [LOA ON 2026 Summer](https://shizukaziye.github.io/loa-on-2026-summer/) — showcase recap

**League of Legends**
- [Champion Pool Coverage](https://shizukaziye.github.io/lol-pool-coverage/) — analyze your pool vs the live meta

**Finance**
- [FIRE Calculator](https://shizukaziye.github.io/fire-calculator/) — financial-independence / early-retirement planner with fixed, historical, and Monte-Carlo modes
